Команды Git
Команда Описание
git init создание нового репозитория
git init создание нового репозитория
git clone url клонирование репозитория
git add file добавление, обновление файла в репозитории
git add . добавляет изменение из рабочего каталога в раздел проиндексированных файлов
git commit -m "comment" создает комментарий к изменению в файле
git commit -a -m "comment" автоматически создаются файлы, которые были изменены и удалены, но новые, о которых вы не сообщили Git, не затрагиваются
git push url обновляет репозиторий
git status выводит статистику изменений каталога
git log отображает отправленные снимки состояния и позволяет просматривать и фильтровать историю проекта, а также искать в ней конкретные изменения
git branch branch_name добавление новых веток
git checkout branch_name позволяет перемещаться между ветками
git checkout -b branch_name создаёт указатель (ветку) на текущую позицию
git remote -
